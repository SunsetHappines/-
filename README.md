# Голосовой музыкальный ассистент (PyQt6 + VLC + YouTube)

Десктоп-приложение на Python с графическим интерфейсом (PyQt6), которое:
- сканирует локальную музыкальную папку и формирует библиотеку в SQLite,
- проигрывает локальные треки через VLC,
- умеет искать треки на YouTube, доставать прямую аудио-ссылку через `yt-dlp` и проигрывать,
- поддерживает голосовые команды с “wake word” **«ассистент»** (SpeechRecognition).

---

## Возможности

- Локальная библиотека:
  - сканирование папок (mp3/wav/flac/m4a),
  - хранение в `music_library.db` (SQLite),
  - быстрый поиск по исполнителю/названию.
- Проигрывание:
  - Play/Pause, Stop, Next/Prev,
  - регулировка громкости.
- YouTube:
  - поиск по запросу,
  - автоматическое извлечение первого найденного видео,
  - получение прямого URL аудио потока через `yt-dlp`,
  - воспроизведение потока через VLC.
- Голосовое управление:
  - включение/выключение режима прослушивания,
  - команды на русском.

---

## Требования

- Python 3.10+ (рекомендуется)
- Установленный **VLC player** (нужны библиотеки VLC для `python-vlc`)
- Рабочий микрофон (для голосового режима)

## Установка

1) Клонировать репозиторий:
bash
git clone https://github.com/SunsetHappines/-.git
cd <repo-name>

2) Создать виртуальное окружение и активировать:
bash
python -m venv .venv
.venv\Scripts\activate

3) Установить зависимости:
bash
pip install -U pip
pip install pyqt6 python-vlc SpeechRecognition requests yt-dlp pyaudio

> Если `pyaudio` не ставится на Windows:  
> - установите готовый wheel (часто проще через `pipwin`), или  
> - используйте альтернативный способ установки PyAudio под вашу версию Python.

## Настройка

В начале файла можно изменить параметры:

- Папки с музыкой:
python
MUSIC_FOLDERS = [r"C:\Users"]

- База данных:
python
DB_PATH = "music_library.db"

- Ключевое слово для активации голосовых команд:
python
WAKE_WORD = "ассистент"

---

## Запуск

Запуск приложения:
bash
python main.py

При первом запуске приложение просканирует папки из `MUSIC_FOLDERS` и создаст/обновит базу `music_library.db`.

---

## Использование

### Локальная музыка
- Слева список треков (из БД).
- Клик по треку — воспроизведение.
- Поиск через строку “Поиск музыки...” ищет по локальной библиотеке.

### YouTube
Если по запросу ничего не найдено локально — приложение:
1) выполняет поиск на YouTube,
2) берет первое найденное видео,
3) получает прямую ссылку аудио через `yt-dlp`,
4) воспроизводит через VLC.

## Голосовые команды

Сначала включите “Голосовой помощник” в интерфейсе.

Формат: произнесите **«ассистент …»**, например:

- **«ассистент включи песню киш лесник»**
- **«ассистент поставь музыку ramstein du hast»**
- **«ассистент пауза»**
- **«ассистент стоп»**
- **«ассистент следующий»**
- **«ассистент предыдущий»**
- **«ассистент громче»**
- **«ассистент тише»**
- **«ассистент громкость 30»**
